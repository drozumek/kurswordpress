Dołączanie plików CSS i JS

Rozbudujemy nasz motyw, poprzez stworzenie dołączenie plików CSS i Javascript. Można to oczywiście zrobić dodając je w nagłówku i stopce strony, jednak jeśli chcemy zrobić to poprawnie i uzyskać wsparcie systemu wordpressa oraz kompatybilność z niektórymi wtyczkami to musimy zrobić to za pomocą haków.

Tworzymy podstawową strukturę katalogów tworząc foldery css i js i umieszczając w nich odpowiednio pliki main.css oraz scripts.js (tutaj nazewnictwo katalogów i plików jest dowolne, ale najczęściej spotyka się takie nazwy katalogów i w kursie też będziemy ich używać).

W terminalu w edytorze za pomocą komend mkdir (make directory) oraz touch stworzymy pliki i katalogi.

mkdir css && touch css/main.css && mkdir js && touch js/scripts.js

Rozbudujemy nasz motyw, poprzez dołączenie PODSTAWOWYCH STYLI!!! ze styles.css Bootstrapa. Można to oczywiście zrobić dodając odpowiednie linki w nagłówku i stopce strony, jednak jeśli chcemy zrobić to poprawnie i uzyskać wsparcie systemu wordpressa oraz kompatybilność z niektórymi wtyczkami to musimy zrobić to za pomocą haków.

Musimy stworzyć plik functions.php, który przechowywać będzie haki oraz nasze funkcje, które muszą zostać wykonane zanim nasza strona zostanie wyświetlona. Dzięki temu możemy w prosty sposób manipulować domyślnym zachowaniem wordpressa w obrębie naszego motywu (każdy motyw posiada osobny plik functions.php).

touch functions.php
